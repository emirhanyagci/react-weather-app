# Emirhan Yagci React Weather App


## SETUP

- Create `.env.local` file and fill in parallel with  [`.env.local.example`]([https://pages.github.com/](https://github.com/emirhanyagci/react-weather-app/blob/main/.env.local.example))

## START
With NPM:

```
$ npm install
```

With Yarn:
```bash
$ yarn add
```
- And ready to see
  
With NPM:

```
$ npm run dev
```

With Yarn:
```bash
$ yarn dev
```
